# Final Project - Fendi Septiawan

Sistem penentuan tempat kerja tukang servis pada jasa perbaikan AC berbasis *online*. Informasi lebih lanjut kunjungi [Sistem Pendukung Keputusan][df1].

### Metode yang digunakan :
  - Entropy (pembobotan)
  - TOPSIS (perangkingan)
